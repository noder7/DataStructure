# Insertion Sort
<pre><code>
the first element is already sorted and all the right side Elements are unsorted,
we need to insert all elements one by one from left to right in the sorted Array.
</code></pre>

### Best Case
<pre><code>
O(n)
</code></pre>

### Average Case
<pre><code>
O(n<sup>2</sup>)
</code></pre>

### Worst Case
<pre><code>
O(n<sup>2</sup>)
</code></pre>
